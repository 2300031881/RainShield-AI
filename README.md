# RainShield-AI
AI-powered parametric insurance for food delivery workers



## Problem
Delivery workers (Zomato, Swiggy) lose 20–30% of their income due to rain and weather disruptions. There is no financial protection for this loss.



## Solution
RainShield AI is an AI-powered insurance platform that:
- Monitors real-time weather conditions
- Detects disruptions like heavy rainfall
- Automatically triggers claims
- Instantly pays workers for lost income



## Target Users
- Food delivery riders (Zomato, Swiggy)
- Depend on daily earnings



## Workflow
1. User registers on the platform  
2. AI calculates weekly premium  
3. System monitors weather API  
4. Rain exceeds threshold  
5. Claim triggered automatically  
6. Instant payout processed  



## Weekly Pricing Model
- Base premium: ₹20/week  
- High-risk zones: ₹40–₹50/week  
- Based on rainfall history and location  



## Parametric Triggers
- Rainfall > 20mm → claim triggered  
- Extreme weather alerts  


##  AI Usage
- Risk-based premium calculation  
- Income loss prediction  
- Fraud detection  


## Fraud Detection
- GPS mismatch detection  
- Duplicate claims prevention
